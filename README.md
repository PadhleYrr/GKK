# 🌿 GKK — Gumasta Krishi Kendra
### Smart Shop Management + AI Krishi Assistant

A complete **offline shop management PWA** for Gumasta Krishi Kendra. Install it like an Android app directly from the browser — no Play Store needed!

---

## 📱 Features

### 🏪 Shop Management
- **New Bill / POS** — Create bills instantly, cash/UPI/udhar payment modes
- **Orders** — Full bill history with filters
- **Products** — Add/edit products with buying price, selling price, margin auto-calc
- **Stock Management** — Track stock, get low stock alerts
- **Returns** — Handle returns, auto-restock

### 💰 Finance
- **Udhar Ledger** — Track customer credit, collect payments
- **Profit & Loss** — Today/Week/Month/All time P&L
- **Reports** — Date range reports with daily breakdown
- **Profit Calculator** — Quick margin calculator

### 🤖 AI Features
- **AI Krishi Assistant** — Ask anything about seeds, fertilizers, pesticides (Hindi + English)
- **Disease Scanner** — Take a crop photo → get instant disease diagnosis + treatment

## 🚀 Setup (Phone Only!)

### Step 1: Create GitHub Account
Go to [github.com](https://github.com) → Sign up (free)

### Step 2: Create New Repository
1. Click **+** → **New repository**
2. Name: `GKK`
3. Set to **Public**
4. Click **Create repository**

### Step 3: Upload Files
1. Click **uploading an existing file**
2. Drag and drop ALL files from this folder
3. Also upload the `.github/workflows/deploy.yml` file
4. Click **Commit changes**

### Step 4: Enable GitHub Pages
1. Go to repo **Settings** → **Pages**
2. Source: **GitHub Actions**
3. Save

### Step 5: Get Your App URL
Wait 2 minutes → Your app will be live at:
`https://YOUR_USERNAME.github.io/GKK`

### Step 6: Install as App on Android
1. Open the URL in **Chrome** on your phone
2. Tap the **⋮ menu** → **Add to Home Screen**
3. Done! GKK is now installed as an app 📱

## 💾 Data Storage
All data stored in **localStorage** — works offline, no internet needed for shop operations. Only AI features need internet.

## 📤 Backup
Go to Settings → Export All Data → saves a JSON backup file.
